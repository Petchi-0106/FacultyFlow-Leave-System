# FacultyFlow – Smart Leave Management System

## 📌 Project Description
FacultyFlow is a Leave Management System developed using Zoho Creator.  
It is designed to manage leave requests of college staff members across different departments with an automated approval workflow.

The application ensures a structured and role-based leave approval process between Staff and Head of Department (HOD).

---

## 🎯 Objective
To automate the leave request and approval process for college faculty members using Zoho Creator and deploy the application via GitHub Pages.

---

## 🏢 Departments Included
- CSE  
- IT  
- ECE  
- EEE  
- Civil  
- Chemical  
- Biomedical  

---

## 👥 User Roles

### 👤 Staff
- Submit leave requests  
- View their leave history  
- Receive approval/rejection notifications  

### 👨‍💼 Head of Department (HOD)
- View department leave requests  
- Approve or Reject leave  
- Send status updates automatically  

### 🛠 Admin
- Manage staff records  
- Manage HOD records  
- View complete reports and dashboard  

---

## ⚙️ Features

- Staff Leave Request Form  
- Automatic email notification to respective HOD  
- Role-based access control  
- Approval / Rejection workflow  
- Leave balance deduction (on approval)  
- Dashboard with leave statistics  
- Department-wise filtering  
- Status tracking (Pending / Approved / Rejected)

---

## 🛠 Technology Used

- Zoho Creator (Application Development)
- Deluge Script (Workflow Automation)
- Email Notification System
- GitHub Pages (Deployment)
